# Roadmap

## Current

- Minimal schema validation.
- Deterministic normalization.
- Deterministic package manifest generation.
- Fixture examples and tests.

## Near Term

- Add adapter examples for common official source formats.
- Add stricter source metadata validation.
- Add JSON Schema export.
- Add municipality identifier guidance.
- Add documented review workflow for data-source pull requests.

## Later

- Track source-specific import adapters by canton and tax category.
- Add richer package metadata for downstream calculator projects.
- Add release checks for reproducible package artifacts.
- Add community-maintained verification notes for official data sources.

This roadmap does not include official tax calculation formulas unless they are implemented from verified official sources and reviewed separately.
